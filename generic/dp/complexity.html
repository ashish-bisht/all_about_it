The Golden Framework
Pure Recursion
Decision Tree
TC = Total nodes in tree
= choicesdepth

SC = O(depth) ← stack only
2 choices (take/skip) → 2^n
3 choices (ins/del/rep) → 3^(n+m)
Depth = max recursion level
Memo / Bottom-Up
Optimized Grid
TC = Unique States × Work/State
= States × O(1) usually

SC = O(states) + O(depth)
Work/state = sirf us node ka kaam
Recursive calls count NAHI hoti
Almost hamesha O(1) per state
Pro-tip: "Work done in ONE state" calculate karte waqt, recursive calls ki deep branching ko count nahi karna. Sirf us node par kya operation ho raha hai — min(), max(), +1 — wo dekhna. Almost hamesha O(1) hota hai.
