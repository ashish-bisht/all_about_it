Pattern Recognition

Sliding Window

  • Use when: "Find max/min subarray of size K"
  • Use when: "Longest substring with X property"
  • Template: Two pointers, expand right, shrink left
  • Key insight: window_sum += arr[r]; window_sum -= arr[l]

Two Pointers

  • Use when: Sorted array, find pairs
  • Use when: In-place modification needed
  • Variants: Start/End, Fast/Slow, Same Direction
  • Key insight: if sum > target: r-- else l++

Binary Search

  • Use when: Sorted data, O(log N) needed
  • Use when: "Find first/last occurrence"
  • Use when: Search in rotated sorted array
  • Key insight: mid = l + (r - l) // 2

DFS / BFS

  • Use DFS: Path finding, exhaustive search
  • Use BFS: Shortest path, level order
  • DFS ‚Üí Stack/Recursion, BFS ‚Üí Queue
  • Key insight: Track visited set!

Dynamic Programming

  • Use when: Overlapping subproblems
  • Use when: Optimal substructure
  • Steps: Define state ‚Üí Base case ‚Üí Transition
  • Key insight: dp[i] = fn(dp[i-1], dp[i-2]...)

Heap / Priority Queue

  • Use when: "Find K largest/smallest"
  • Use when: Merge K sorted lists
  • Min heap for K largest, Max heap for K smallest
  • Key insight: Maintain heap of size K

Monotonic Stack

  • Use when: "Next greater/smaller element"
  • Use when: Span problems, histogram area
  • Maintain increasing or decreasing order
  • Key insight: Pop while stack[-1] < current

Backtracking

  • Use when: All permutations/combinations
  • Use when: Constraint satisfaction (N-Queens)
  • Template: Choose ‚Üí Explore ‚Üí Unchoose
  • Key insight: Prune early with constraints
